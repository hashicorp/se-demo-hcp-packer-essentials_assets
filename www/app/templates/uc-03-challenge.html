AWS vs. Azure

You can deploy images in both AWS and Azure, but the variable names may differ between the two.

Therefore, you will need to use your AWS Packer template as a guide to create an Azure Packer template.

This challenge will guide you through the process of converting from AWS syntax to Azure syntax.
